# 招聘数据爬取与分析(人工智能相关方向)
## 概述
项目旨在给人工智能相关方向的同学在求职时提供尽可能详尽有用的信息,以便于进行岗位选择与技能完善
项目分为数据爬取与数据分析两部分
## 数据爬取
数据爬取了 智联 boss直聘 猎聘三个网站上的 有关 人工智能 AI算法 NLP算法 自然语言算法 数据挖掘 数字人 CV这几个方向上的有关 岗位名称 区位 薪资 工作年限 学历 关键词 公司名 企业类别 企业规模 岗位要求 更新时间 这几方面的信息
